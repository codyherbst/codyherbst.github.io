Week 1 Blog Post

What did you learn about that helped you understand more about design?

Learning bootstrap has been the single most important lesson when it comes to understanding design. I in no way would have been comfortable doing design if only using CSS and html was a thing.

What do you want to continue learning about that helps you understand design better?

I want to continue to learn how to design for mobile first. It seems to me developing with the screen set to mobile at the beginning will allow me to focus better on designing a more responsive website.

What Design aspects are important to you as a user and as a developer?

The above answer sort of answers this question. Mobile first seems to be the most important consideration when designing a website for a user. Accessibility seems to be the most important aspect as a user however. If you have to jump through hoops on a website to do something then that user will likely bounce and not stay.

What is the number one thing that held you back this week? Why?

Without exposing myself too much, Dealing with personal life issues has been the biggest hold up. Everyday is impacted by it but I only look to persevere, even if I cannot perform to the best of my ability. Doing this interests me more than anything I have done in the past.

What is one thing you would like to get better at in bootcamp?

I feel I need to get better at design. It is the one thing so far that I don't feel very confident in nor interested in.
