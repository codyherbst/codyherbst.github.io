Weather App

Previous Next

The weather app is an interesting project. It involves learning how to effectively use an API to display weather data using the Open Weather API.

These are some shortcomings that I have not overcome yet:
  • Incorrect Data gives a basic alert box
  • Geolocation is not implemented
  • The design is mediocre

The basic data alert box is a quick fix and one I hope to implement quickly. However, I am not confident in setting up the geolocation. I have already tried and failed to understand how it works. The parameters it takes/gives are confusing for me at the moment. As for the design, I do not intend to improve on that immediately. Design is not my strong suit as I find it difficult to do something that looks good without a template.

I have thoroughly enjoyed this project despite the above shortcomings. A future project of mine is to create a news conglomerate website that auto-summarizes articles. This would be done using an API from Stanford's NLP resource. Without learning how to use an API this project would be fruitless.
